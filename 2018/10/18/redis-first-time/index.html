简述redis

redis 即remote dictionary server, 主要用于高速缓存, 去优化系统的的性能.

redis通常放置体量小的热点数据, 体量大的数据放置在MySQL/ MangoDB 更合适.

安装redis

1.在redis.cn中可以找到下载源文件链接wget <download site>

2.对下载的文件解压缩 gunzip <file>

3.对解压缩的文件进行解归档 tar <file>

4.进入解归档的文件, 发现它已经有了makefile 的文件, 就不用进行进行配置, 直接构建安装 make && make install
